Rewired State Welcome Trust Open Science Hack 2012

The Brief

For this hack I decided to use data from HESonline and in particular maternity data for 2011-12. AS the data came as an Excel spreadsheet, the first task was to extract it from the spreadsheet and store it into a more accessible format. I chose to store it as JSON objects in Riak. I then built a small Django application that creates charts based on the data. The file structure reflects this:

  • The dataload directory contains a command line Java application that uses Apache POI to read the Excel file and then loads the data into a Riak database.
  • The dataview directory contains a django application that uses the Riak data to product graphs.
